# AntiDependency AI
An AI that strengthens human thinking instead of replacing it.
An AI designed not to replace thinking —
but to strengthen it.

Instead of directly answering complex questions,
the AI guides users through reasoning pathways
using Socratic questioning and critical thinking.

## Tech Stack
- React
- Node.js
- Express
- OpenAI API

## Philosophy

Most AI systems optimize for convenience.

AntiDependency AI optimizes for human intellectual growth.
